# ap-node-bin

node bin utility

## Installation

```
npm i -g ap-node-bin
```

## Usage

You just need to add all commands to bin field in package.json like below.

```
"bin":{
    "your-command-1":"bin/your-command-file1.js",
    "your-command-2":"bin/your-command-file2.js",
}
```

## Run command in terminal

```
npx ap-node-bin
```
